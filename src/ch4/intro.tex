\chapter{Results\label{ch:ch4}} 
This chapter presents the numerical results, obtained using the post-processing
tool \gls{HAOT} introduced in \Cref{ch:ch3}.  The results presented in this chapter investigate three different levels of
fidelity (these fidelities are highlighted in orange in \Cref{fig:gdFidelities}).
\begin{itemize}
    \item \Cref{sec:lowFidelity} covers the low-fidelity enthalpy model
        (temperature-dependent \gls{GD}), high fidelity flow.
    \item \Cref{sec:medFidelity} covers the medium-fidelity enthalpy model
        (species-specific \gls{GD}), low fidelity flow. 
    \item \Cref{sec:highFidelity} covers the highest-fidelity enthalpy model (internal state-specific, species-specific \gls{GD}).
\end{itemize}

\Cref{sec:lowFidelity,sec:medFidelity,sec:highFidelity} will be using the same ground
equilibrium constants. The
