\chapter{Aero-Optics Tool\label{ch:ch4}}
An open-source tool, \gls{HAOT}, has been developed to perform this analysis.
The tool is published on \href{https://pypi.org/project/haot/}{PyPI}, with its
source code hosted on \href{https://github.com/mliza/HAOT}{GitHub}, and
the documentation provided on \href{https://haot.readthedocs.io/en/latest/}{Read the Docs}. 
The current version of \gls{HAOT} includes five modules:

\begin{itemize} 
\item \textbf{Aerodynamics:} Implements commonly used aerodynamic functions.
\item \textbf{Optics:} Provides equations used in aero-optics and hypersonic
    flow analysis.
\item \textbf{Quantum Mechanics:} Offers commonly used quantum mechanical
    functions relevant to hypersonic flow.
\item \textbf{Constants:} Contains various constants, primarily
    spectroscopy-related used in optics.
\item \textbf{Conversion:} Provides a suite of conversion functions used in
    this module. 
\end{itemize}

Validation results are provided in subsequent sections. The constants and
conversion functions were validated through visual inspection. The primary
objective of \gls{HAOT} is to enable researchers to perform optical analysis
using \gls{CFD} results. \gls{HAOT} is an open-source tool designed with the
intention of allowing other developers to expand its current functionalities.
The goal of the tool is to serve as a \textit{software blueprint} that provides
a foundational framework for researchers, particularly in academia, where
software tools are often poorly written, inadequately tested, insufficiently
documented, and underdeveloped \cite{Robillard1998,Liu2008}. Comprehensive
documentation for the tool, including usage instructions with or without
\gls{CFD} data, can be found on the
\href{https://haot.readthedocs.io/en/latest/}{documentation page}. The
\gls{HAOT} tool has a GitHub pipeline that automatically runs unit tests to
ensure that the functions developed do not break when changes in the code are
submitted.
