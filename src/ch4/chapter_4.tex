\chapter{Aero-Optics Tool\label{ch:chapter4}}
An open-source tool, \gls{HAOT}, has been developed to perform this analysis.
The tool is published on \href{https://pypi.org/project/haot/}{PyPI}, the
source code is available on \href{https://github.com/mliza/HAOT}{GitHub}, and
the documentation can be found on
\href{https://haot.readthedocs.io/en/latest/}{Read the Docs}. The current
version of \gls{HAOT} includes five modules:

\begin{itemize}
    \item \textbf{Aerodynamics:} Implements commonly used aerodynamic functions.
    \item \textbf{Optics:} Implements optics equations used in hypersonic flow analysis.
    \item \textbf{Quantum Mechanics:} Implements commonly used quantum mechanical functions for hypersonic flow applications.
    \item \textbf{Constants:} Includes various constants, primarily spectroscopy constants, used across the different modules.
    \item \textbf{Conversion:} Provides several conversion functions used within the package. \end{itemize}
