\section{Aerodynamics\label{ch4:aerodynamics}}
The \gls{HAOT}, \myPythonInline{aerodynamics} module contains the following
functions. Refer to the
\href{https://haot.readthedocs.io/en/latest/modules/aerodynamics.html}{aerodynamics documentation}
to see the implementation of each function.
\begin{itemize}
    \item \myPythonInline{sutherland_law_viscosity}  
    \item \myPythonInline{sutherland_law_conductivity}  
    \item \myPythonInline{air_atomic_molar_mass}
    \item \myPythonInline{speed_of_sound}
    \item \myPythonInline{isentropic_relations}
    \item \myPythonInline{normal_shock_relation}
    \item \myPythonInline{oblique_shock_angle}
    \item \myPythonInline{oblique_shock_relations}
\end{itemize}

Unit test results for the \myPythonInline{aerodynamics} module in the
\gls{HAOT} package are provided below. \\

packages and compares the results against the
\href{https://ambiance.readthedocs.io/en/latest/}{Ambiance} package, which is
an implementation of the \textit{ICAO Standard Atmosphere 1993} manual \cite{InternationalCivilAviationOrganization1993}.
