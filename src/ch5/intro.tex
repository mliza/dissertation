\chapter{Results\label{ch:ch5}} 

This chapter presents the numerical results, obtained using the post-processing
tool \gls{HAOT} introduced in \Cref{ch:ch4}. All the results in
this chapter will use the same equilibrium polarizability constants. \\

The equilibrium polarizability constants for neutral and ion species are
provided on \Cref{tab:equilibriumPolarizabilityNeutralSpecies} and
\Cref{tab:equilibriumPolarizabilityIonSpecies}.
