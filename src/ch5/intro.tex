\chapter{Numerical Results\label{ch:chapter5}} 

This chapter will discuss some numerical results, all these results used the
post-processing tool discussed in Chapter \ref{ch:chapter4}. All the results in
this chapter will use the same polarizability constants. \\

polarizability constants (ion species) used for this study are provided on
\TabRef{tab:ionizationStudyEquilibriumPolarizability} and
\TabRef{tab:ionizationStudyNonequilibriumPolarizability}.
