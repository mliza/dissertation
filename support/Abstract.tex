 \chapter*{Abstract}
Being able to control the plasma state and its evolution in real time is crucially important for magnetic confinement nuclear fusion stability. However, the complex physics describing the plasma state requires an extensive amount of computational power and a long computational time. An alternative for this conundrum is to develop a model that simplifies the physics, lowering the running time.\\

Experimental data combined with a model that predicts the transport behaviour of a tokamak (COMPASS), allows to accurate predict the state of the system. An accurate prediction from the plasma state allows the development and integration of a closed loop control system. \\

A simulation of a sawtooth instability (plasma instability) using real gathered data from the COMPASS’s sensors; is developed in this thesis. This simulation uses a 1D Rapid Plasma Transport (RAPTOR) code.\\

This code solves two coupled 1D partial differential diffusion equations for the electron temperature and the poloidal magnetic flux. The RAPTOR code combined with collected data allowed to calculate the critical magnetic shear for COMPASS tokamak.
